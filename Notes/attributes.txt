head tags:
---------------------------------------------------------------------------------------------------------------------------------------------
1.this is for the search engine to analyse the webpage for better seo effects (for search engine)
	<meta name='keywords' content='power pc, download, repair services'/>
2.this is also for the search engine wich give the description of the page (for search engine)
	<meta name='description' content='this page is a test page and this content should hold the description of the page'/>
3.this will tell us who is the owner of the page
	<meta name='author' content=''/>
4.this will give us the ownership of the page
	<meta name='copyright' content=''/>
5.this is to describe back-end technology (optional)
	<meta name='generator' content=''/>
6.this will decide if the website is open to search engines or not 
	<meta name='robot' content='all(for best seo effect) or none(for internal system pages)'/>
7.this reload the page after 5s and take us to google.com (for search engine and browser)
	<meta http-equiv='refresh' content='5;http://www.google.com'/>
8.this is for multilanguage use in websites
	<meta http-equiv='content-type' content='text/html;charset=utf-8'/>

Body tags:
---------------------------------------------------------------------------------------------------------------------------------------------
1.<p>Paragraph tag</p> (starts in a new line)

inline CSS: (stored in the tag)
ex: <p style='color:red(or color code);font-weight:bold(or normal, bolder, light, lighter);'>This is a red bold paragraph</p>

internal CSS: (stored in the head area)
<style>
	*{color:red}			//universal selector
	p{color:red}			//type selector (p is the paragraph tag[this will affect all p tags])
	.class_name{}			//.class selector (we must assignt a class name in the tag we want to style ex: <p class='red'>texr</p>)
	#ID_name{}			//#ID selector (we must assignt a ID name in the tag we want to style ex: <p id='ID+_green'>texr</p>)
</style>

2.<span>this tag is an inline tag wich means that it does not occupy all the line but only the necessary space for the text inside it</span>

++Example of style tags:
	{background:blue}
	{font-size:100px}
Note : Block tag's width and height can be controlled by CSS, but Inline tag's width and height are decided by it's contents.

3.<h1></h1> till <h6></h6> (block tag)

we can use em instead of px for font sizes.
